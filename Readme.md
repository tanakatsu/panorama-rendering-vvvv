# Panorama rendering sample (vvvv)

This is a panorama rendering sample for vvvv.

### Requirements
- [vvvv](https://vvvv.org/)

### How to use

1. open v4p file

##### Keyboard control

- Up, down, left, right: camera control

##### Mouse control

- Play button (Right click): start playing movie 
- Do Seek button (Right click): replay
